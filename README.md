# Semester Study Assistant

Semester Study Assistant is a Java-based Command Line Interface (CLI) application
designed to help students manage their academic workflow.

It provides:
- Subject-wise study guidance based on syllabus
- Suggested revision topics
- Practice-style questions for self-learning
- A simple interactive interface using keyword detection
